키워드 입력 순위체크 결과 성공지표

키워드 입력

상품등록 후 등록하신 키워드의 네이버 노출 순위를 분석하기 위한 키워드 등록 페이지 입니다.

사업자
판매채널
키워드
상품명
mid값

mid 값이란?
스토어팜에 등록한 상품 url의 고유 number로 순위체크를 할 수 있는 id 입니다.

mid 추출방법
1. 등록된 상품의 제목(상품명)을 복사
2. 네이버쇼핑 검색창에 복사한 상품명 검색
3. 검색된 상품명 우측마우스 클릭 후 링크 주소 복사
4. 메모창에 링크 붙여넣기
5. 끝부분에 "~mid=A&~" 중 A영역이 mid 값

자동체크
사업자 판매채널 키워드 제품명 mid값 자동여부

성공지표

지정하신 날짜에 맞춰 네이버 쇼핑 검색에 키워드를 입력 / 검색 했을 때,
CPC 광고를 제외한 1페이지 중 10위건 내에 노출된 상품 리스트의 정보를 노출합니다.

`

월간 조회수인데 왜 일별로 지정이 되어있나요?

TripleS에서 지정한 <월간조회수>는 지정된 날짜를 기준으로 한달 전 조회수이며,
--------이조회수의 예시로는--------

ex01) 20xx/11/12의 월간 조회수 =20xx/10/12조회수 ~ 20xx/11/12조회수의 총 값

ex01) 20xx/11/13의 월간 조회수 =20xx/10/13조회수 ~ 20xx/11/13조회수의 총 값

---------------------------------------
으로 날짜별로 갑자기 키워드가 급상승 또는 급하락 할 경우 일별로 나열이 되지만 큰 숫자 차이가 일어날 수 있습니다.
